# AI Saathi Frontend

Welcome to the AI Saathi Frontend repository. This project serves as the premium, cinematic, and highly interactive user interface for the AI Saathi application. It features a completely custom layout, high-fidelity micro-interactions, responsive design, and cutting-edge emotional storytelling capabilities.

## Features

- **Cinematic Landing Page:** A visually stunning homepage that leverages Framer Motion and custom SVG components (such as the organic Root Architecture and Breathing Emotion Orb) to bring the brand to life.
- **Robust Component Architecture:** Modular architecture built entirely with React, TypeScript, and Vite.
- **Tailwind & Custom CSS:** Built with Tailwind properties, complemented by custom Glassmorphism and Claymorphism CSS extensions for a premium visual identity.
- **Emotion Engine Visualization:** Real-time state-driven UI layers that narrate the AI's internal process and current emotional context.

## Tech Stack

- **Framework:** Vite + React + TypeScript
- **Styling:** Tailwind CSS + Shadcn UI
- **Animations:** Framer Motion
- **Tooling:** ESLint, PostCSS, Prettier

## Running Locally

1. Setup environment variables (create a `.env` file referencing any needed Supabase or backend variables).
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development frontend server:
   ```bash
   npm run dev
   ```

## Development

The main entry point for the application routing is `src/App.tsx`, where all protected endpoints and primary layouts are orchestrated. Our visual design foundation is maintained in `src/index.css`.
